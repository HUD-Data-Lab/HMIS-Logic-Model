# HMIS Demo Ontology

## Load in Protégé

1. Open Protégé.
2. Load `demo-ontology.ttl` as your base ontology.
3. Load `demo-shapes.ttl` in the SHACL tab.
4. Load `demo-data.ttl` in the Individuals tab.
5. Run SHACL validation to see errors.
6. View mappings in OWL tab (e.g., Male → Man).

## What’s Demonstrated

- Conditional validation by project type
- Date logic (Start ≤ Exit)
- Age/Veteran logic
- Response mapping and retirement
